Fonctionnalités

  • Interface élégante et intuitive
  • Navigateur Edembra
  • IA locale personnalisable
  • Gestionnaire de fichiers Orion
  • Lecteur M3U et ACSCAM
  • Editeur de code Edembra Renoscode
  • Fla pour réduire la latence
  • Metal3Thruee pour la gestion du système
  • MTerminal avec langage Deslang
  • OneStore pour des applications fiables

Remarque : Toutes ces fonctionnalités ne seront pas disponibles immédiatement. Le système pourrait être prêt autour du 24 février.

Logiciels intégrés

  • Orion : Gestionnaire de fichiers avancé
  • Lecteur M3U : Streaming multimédia
  • ACSCAM : Transformation de la caméra
  • Edembra Renoscode : Éditeur de code IA
  • Fla : Optimisation de la latence
  • Metal3Thruee : Outil de gestion
  • MTerminal : Multiterminal
  • OneStore : Accès aux applications fiables
  • Éditeur de texte : Versions Pro et Pulse +
